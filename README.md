**🏥 Healthcare Management System**
**📌 Description**
This is a web-based Healthcare Management System designed to manage patients, hospitals, and appointments efficiently.
It helps users easily access hospital details, improve communication, and simplify healthcare services through a clean and responsive interface.
**🚀 Features
✨ Core Features**
👤 User registration and login system
🏥 Add and manage hospitals
📊 Simple dashboard for data tracking
📱 Responsive and clean UI design
📞 Hospital and doctor contact information
🔄 Upcoming Features

**⚙️ How to Run**

# 2️⃣ Move into project folder
cd healthcare-system

# 3️⃣ Install dependencies
pip install -r requirements.txt

# 4️⃣ Run the application
python app.py

# 5️⃣ Open in browser
http://127.0.0.1:5000
